# Wine Review Rating Predictor AI 🍷

An end-to-end Machine Learning pipeline and Streamlit application that predicts wine expert review scores based on pricing, geographic origin, grape variety, and text-based tasting descriptions. Powered by the Kaggle Wine Reviews dataset.
